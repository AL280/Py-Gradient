## Usage/Examples

```python
from pystyle import *
from colorama import Back, Fore, Style


banner = r'''
 █████╗ ██╗     
██╔══██╗██║     
███████║██║     
██╔══██║██║     
██║  ██║███████╗
╚═╝  ╚═╝╚══════╝                
'''
System.Clear()
Anime.Fade(Center.Center(banner), Colors.purple_to_blue, Colorate.Vertical, interval=0.030, enter=True)
```
